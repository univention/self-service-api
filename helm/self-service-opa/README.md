# self-service-opa

OPA adjusted for the self service API PoC

- **Version**: 0.1.0
- **Type**: application
- **AppVersion**: 0.51.0
-

## Introduction

This chart does install the self-service OPA.

The self-service OPA is a fully stateless component, no special handling of persistent
volumes or similar is needed.
